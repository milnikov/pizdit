# Claim Rating Rules

| Rating | Meaning |
|---|---|
| TRUE | The claim is strongly supported by reliable evidence |
| MOSTLY_TRUE | The claim is mostly supported but may miss some context |
| MIXED | The claim contains both accurate and problematic elements |
| MISLEADING | Selective framing, missing context, or distorted interpretation |
| UNSUPPORTED | Not enough evidence to support the claim |
| FALSE | Reliable evidence contradicts the claim |
| UNVERIFIABLE | Cannot be checked with available reliable sources |
